## 專案介紹
這是**五張撲克**換牌遊戲：
* **可押注**
* **可換一次牌**
* **可重製遊戲**<br><br>

## 遊玩方法
1.  **輸入押注**：輸入金額後按下 **[押注]** 此時總資金會預先扣除下注額，並在第二格確認已投注金額。
2.  **開始發牌**：按下 **[發牌]** 按鈕，系統將隨機分配五張牌到牌桌上。
3.  **挑選換牌**：點擊牌面上想更換的牌（被選中的牌會被翻到背面），接著按下 **[換牌]**。
4.  **結算結果**：按下 **[判斷牌型]**，系統將自動比對賠率表並計算你的最終得分。<br><br>

為了確保遊戲邏輯嚴謹，本程式在處理「總資金」與「押注」時採用了分段計算機制：
1. **押注階段**：
   - 當玩家輸入金額並點擊押注時，**「總資金（第一格）」**會立即扣除該筆押注金額，模擬現實中先下注的行為。
   - 同時，**「總資金（第二格）」**會同步顯示目前已投入的押注金額，方便玩家確認當前成本。

2. **結算階段**：
   - 在按下「判斷牌型」按鈕前，總資金維持扣除後的狀態。
   - 待牌型判斷完成後，程式才根據賠率表計算獎金，並將獲勝金額（或返還金額）重新加回至「總資金」。

> [!IMPORTANT]
> 這種「先扣除、後結算」的設計，能有效防止玩家在未完成遊戲前重複下注或產生金額計算錯誤。
<br><br>
## 快捷鍵說明 
在測試程式時，可以使用以下快捷鍵快速產生對應手牌，用於驗證賠率與牌型判斷是否正確：

* `Q`: **同花大順** 
* `W`: **同花順** 
* `E`: **同花** 
* `R`: **四條** 
* `T`: **葫蘆** 
* `Y`: **三條**
> [!TIP]
> 這些快捷鍵主要用於 **Debug 測試**，確保每一種牌型的加成邏輯都能準確執行。
